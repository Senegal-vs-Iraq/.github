## Сенегал — «Львы Теранги»

<a href="#">Сборная Сенегала</a> — действующий **чемпион Африки (АФКОН)** — разочаровала на старте ЧМ 2026. Под руководством **Папе Тьява** команда уступила Франции 1:3 и Норвегии 2:3, пропустив по 3 гола в каждом матче. Оборонительные ошибки дорого обошлись. ## Тактика

**Сенегал:** Задача — **забить как можно больше** для улучшения разницы. Атака через фланги: Сарр (право) + Мане (лево) тянут на скорость против крайних защитников Ирака. Джексон создаёт хаос в штрафной. Камара организует переходы.

**Ирак:** Без Хусейна нападение буквально обезглавлено. Аль-Хамади против Кулибали — бесперспективная дуэль. Зидан Икбал — единственный реальный вариант для контратак. Ирак скорее всего будет играть от глубины.
